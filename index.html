Features

Everything you need
for time arithmetic

Built for developers, project managers, freelancers, and anyone who works with time.

⏱

Multi-operand Calculator

Chain multiple time values with + and − operators. Build complex expressions step by step and see the full equation before calculating.

💬

Natural Language Input

Type 3h 23m 44s, 1d 4h, or 2y 3mo 1w — no rigid format required. Mix and match any supported units freely.

💼

Work Time Mode

Instantly convert durations to work days, weeks, and months using an 8h/day · 5d/week · 160h/month reference — perfect for billing.

🔢

5 Output Modes

Switch between Clock, Natural, Work, Hours-only, and All-formats views. Your preferred mode is saved automatically across sessions.

⌨️

Keyboard First

Press + to commit and add, − to subtract, Enter to calculate, ⌫ to undo, and Esc to clear — never touch the mouse.

🛡️

Fully Private

Zero data collection. No network requests. Your mode preference is stored locally in your browser only. No accounts, no tracking, ever.
